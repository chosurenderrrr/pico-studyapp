# SPEC - 技術仕様・要件定義

---

## 機能A: カレンダーメモのタイトル/内容分離

### データ構造変更
`calendar.json` の各日エントリを拡張：
```json
{
  "2026-03-08": {
    "memo": "(後方互換用・既存データのみ)",
    "memoTitle": "タイトル文字列",
    "memoContent": "内容文字列",
    "sessions": [...]
  }
}
```
- 既存データ（`memo` のみ）は `memoTitle: ""`, `memoContent: memo` として読み込む

### カレンダーセル表示
- 現在の●ドット（`.cal-dot`）は **削除**
- 代わりに `memoTitle` を日付セルの下にテキスト表示
- 表示スタイル：フォントサイズ 8px、最大幅はセル幅に合わせて overflow hidden + text-overflow ellipsis
- タイトルが空の場合は何も表示しない

### カレンダー詳細エリア
- 入力フィールドを2つに分ける：タイトル（1行）＋内容（複数行）
- `Cal.saveMemo()` を `Cal.saveCalendarMemo()` に改修（タイトル・内容を個別保存）

## 機能B: 学習統計の拡充

### 計測データ

#### セッション時間
- `Quiz.start()` 時に `Date.now()` で開始時刻を記録
- `Quiz.finish()` 時に経過秒数を計算
- `study-log.json` の各エントリに `duration`（秒）を追加

#### 問題ごとの解答時間
- `Quiz.render()` 時に `Date.now()` で各問題の開始時刻を記録
- `Quiz.answer()` 時に経過秒数を計算
- セッション内の問題時間を `{ num: 解答秒数 }` の形式で保持
- `study-log.json` の各エントリに `problemTimes: { "1": 12, "2": 8, ... }` を追加

#### quiz-stats.json への追記
- 各問題キーに `totalTime`（累計解答時間・秒）と `answerCount`（解答回数）を追加
- `avgTime = totalTime / answerCount`（表示時に計算）
- 既存エントリとの後方互換：フィールドがない場合は 0 として扱う

### 表示場所

#### 統計画面（view-stats）への追加
1. **全体カード** に以下を追加：
   - 総勉強時間（時:分表示）
   - 連続学習日数（ストリーク）
2. **セット別カード** に以下を追加：
   - セッション平均時間（分）
3. **問題別ランキングカード**（各セット展開時）：
   - 「最も時間がかかった問題」上位3件（問題番号 + 平均解答時間）
   - 「苦手問題スコア」上位3件（正答率が低く解答時間が長い問題）

   ※苦手スコア = `(1 - 正答率) * 0.7 + (avgTime / maxAvgTime) * 0.3` で算出

#### カレンダー詳細エリアへの追加
- 学習記録セクションに「セッション時間: X分」を追加表示

### 連続学習日数（ストリーク）算出ロジック
- `study-log.json` の date フィールドを走査
- 今日から逆順に連続している日数をカウント
- `Stats.calcStreak()` として実装
